# 需求工单管理系统 - H5移动端原型

## 📱 项目简介

这是一个**需求工单管理系统**的H5移动端高保真原型，采用纯HTML + Tailwind CSS + JavaScript开发，无需后端支持，可直接在浏览器中打开运行，用于产品演示和交互体验。

## ✨ 核心功能

### 1. AI数据寻源助手（默认首页）
- 🤖 智能问答：通过自然语言对话快速查找数据资源
- 💡 快捷提问：预设常见问题，一键快速查询
- 📊 资源推荐：自动匹配并展示合适的数据资源（参照实际业务格式）
- 📋 资源详情：包含名称、描述、来源平台、资源类型、更新频率、标签
- 📜 历史记录：查看所有历史问答记录
- ➕ 新会话：随时开启新的对话

### 2. 我的需求
- 📑 需求列表展示（卡片式）
- 🔍 搜索需求（支持标题、编号）
- 🏷 状态筛选（全部、待处理、进行中、已完成）
- ➕ 新增需求（弹窗表单）

### 3. 需求详情
- 📝 需求基本信息
- ⏱️ 处理进度时间轴
- 💬 操作功能（处理需求、关闭需求）

## 🎨 界面设计特点

### 顶部导航栏
- 采用渐变色设计（紫色渐变）
- 包含返回按钮、标题、新会话/历史按钮
- 统一的视觉风格，符合企业级应用标准

### 数据资源卡片（参照截图设计）
```
┌────────────────────────────┐
│ 实时线索行为数据    [数据集] │
│                            │
│ 用户在产品内的所有行为数据， │
│ 包括页面浏览、点击、表单    │
│ 提交、按钮点击等全量行为日志 │
│                            │
│ ━━━━━━━━━━━━━━━━━━━━━━━━ │
│ 📊 神策数据    🔄 实时     │
│                            │
│ [用户行为] [埋点数据]       │
└────────────────────────────┘
```

卡片包含：
- **名称**：资源的名称（如：实时线索行为数据）
- **描述**：详细的资源描述
- **类型标签**：数据集/标签数据/主数据/报表
- **来源平台**：神策数据/数据中台/CRM系统等
- **更新频率**：实时/T+1/每日
- **标签**：相关的业务标签

## 🚀 快速开始

### 方式一：直接打开（推荐）
1. 双击 `index.html` 文件
2. 在浏览器中打开即可体验

### 方式二：使用本地服务器
```bash
# 使用Python启动简单服务器
python -m http.server 8080

# 然后在浏览器访问
# http://localhost:8080
```

### 方式三：使用VS Code Live Server
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🎨 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架（CDN引入）
- **JavaScript** - 交互逻辑
- **FontAwesome 6** - 图标库
- **LocalStorage** - 历史记录持久化

## 📱 页面功能详解

### AI数据寻源助手
- **智能对话**：类似ChatGPT的对话界面
- **资源匹配**：根据关键词智能匹配数据资源
- **资源展示**：参照实际业务需求的卡片格式
  - 名称、描述、来源平台、资源类型、更新频率、标签
- **快捷问题**：预设常见问题，方便快速查询
- **历史记录**：右侧滑出菜单，查看所有历史对话
- **新会话**：点击顶部"新会话"按钮，开启新的对话

### 我的需求
- **新增需求**：点击右上角"+"按钮，弹出创建需求表单
- **需求搜索**：实时搜索需求标题和编号
- **状态筛选**：通过Tab切换不同状态的需求

### 需求详情
- **信息展示**：完整的需求信息和描述
- **状态展示**：在基本信息标题处显示当前状态
- **进度时间轴**：展示需求处理的全流程
- **操作按钮**：处理需求、关闭需求

## 🔗 页面跳转关系

```
AI助手 (index.html) ←→ 我的需求 (list.html)
                              ↓
                         需求详情 (detail.html)
```

## 💡 交互说明

1. **顶部导航**：统一的渐变色导航栏，包含返回、标题、功能按钮
2. **AI对话**：输入问题后自动智能匹配资源
3. **新会话**：点击顶部"新会话"按钮清空当前对话
4. **历史记录**：点击右上角历史图标查看
5. **新增需求**：在我的需求页点击右上角"+"按钮
6. **表单验证**：必填项验证
7. **弹窗交互**：详情页的处理等弹窗可以打开和关闭
