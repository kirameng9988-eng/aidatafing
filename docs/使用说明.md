# 🎯 快速使用指南

## 📱 如何打开原型

### 方法1：直接双击（最简单）
1. 找到项目文件夹中的 `index.html`
2. 双击打开
3. 在浏览器中即可体验完整原型

### 方法2：拖拽到浏览器
1. 打开Chrome/Edge/Firefox浏览器
2. 将 `index.html` 文件拖拽到浏览器窗口
3. 自动打开原型页面

---

## 🎨 功能演示流程

### 第一步：体验AI数据寻源助手（默认首页）

打开后，您会看到AI数据寻源助手界面：

#### 1.1 快捷提问
点击预设的快捷问题按钮：
- "我想找用户行为数据"
- "需要销售相关的数据表"
- "有哪些财务数据可用？"

#### 1.2 自由提问
在输入框中输入问题，例如：
- "帮我找用户订单相关的数据"
- "我需要查看交易流水"
- "有没有客户信息的表？"

#### 1.3 查看资源推荐
AI会智能匹配并展示数据资源卡片，包含：
- 资源名称
- 数据库.表名
- 数据类型
- 主要字段
- 更新频率
- 数据质量
- 负责团队

#### 1.4 历史记录
- 点击右上角"历史"图标
- 查看所有历史对话
- 点击可重新加载历史对话

---

### 第二步：体验工单管理

点击底部导航的"工单管理"Tab

#### 2.1 查看工单列表
- 浏览所有工单卡片
- 查看工单状态和进度

#### 2.2 搜索工单
在搜索框中输入：
- 工单标题关键词
- 工单编号

#### 2.3 筛选工单
点击Tab切换不同状态：
- 全部
- 待处理
- 进行中
- 已完成

#### 2.4 新增工单
1. 点击右上角"+"按钮
2. 填写工单信息：
   - 工单标题（必填）
   - 工单描述
   - 优先级（高/中/低）
   - 工单类型
3. 点击"提交工单"
4. 查看成功提示

#### 2.5 查看工单详情
1. 点击任意工单卡片
2. 查看完整信息
3. 查看进度时间轴
4. 尝试操作：
   - 处理工单
   - 添加评论
   - 关闭工单

## 📋 常见问题

### Q: 数据是真实的吗？
A: 当前是原型演示，使用的是模拟数据。正式环境会对接真实的元数据管理系统。

### Q: AI对话是真实的吗？
A: 当前是基于关键词匹配的模拟对话。正式环境会接入大模型API，提供更智能的交互。

### Q: 历史记录会丢失吗？
A: 历史记录保存在浏览器LocalStorage中，清除浏览器数据会丢失。正式环境会保存到服务器。

### Q: 可以在手机上查看吗？
A: 可以！直接在手机浏览器中打开即可，完全适配移动端。

## 🔧 演示技巧

1. **推荐使用Chrome浏览器**：体验最佳
2. **打开开发者工具**：F12 → 切换到移动端模式，可以看到不同设备的效果
3. **清空历史记录**：在浏览器控制台输入 `localStorage.clear()` 即可清空历史
4. **修改模拟数据**：编辑 `src/data/mock.js` 可以修改工单数据
